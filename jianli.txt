项目名： GoAgent — 基于 Go 的 ReAct AI Agent 框架
描述：

独立设计并实现了一个 AI Agent 框架，基于 ReAct 模式（Reasoning+Acting）自主完成多步骤任务。
使用 Go 原生 goroutine 并发执行多工具调用，相比串行方案减少等待时间；
基于 Gin 框架封装 HTTP API，支持多用户并发会话管理（sync.RWMutex）；
对接 DeepSeek LLM 的 Function Calling 协议，实现可插拔的 Tool 扩展机制。

技术栈： Go、Gin、LLM Function Calling、ReAct、goroutine、REST API
